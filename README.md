## Rake.el

**Rake.el** is a package for running rake tasks in Emacs.

* runs rake command using zeus, spring or bundler
* caches the tasks

## Installation

## Usage

Do `M-x rake-compile` to run a rake task.

## Contribution

Install [cask](https://github.com/rejeep/cask.el) if you haven't
already, then:

```bash
$ cd /path/to/rake.el
$ cask
```

Run all tests with:

```bash
$ cask exec ecukes
```
